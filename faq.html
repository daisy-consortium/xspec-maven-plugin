General

What is XSpec?
XSpec is a Behavior Driven Development (BDD) framework for XSLT and XQuery. For more information, please refer to the XSpec site.
What goal does the XSpec Maven Plugin offer?
The plugin provides the gal xspec:test to run XSpec XSLT tests.
What version of XSpec is used by the plugin?

Version 0.3.0 of XSpec is used by version 1.0.0-SNAPSHOT of the XSpec Maven Plugin, with only the following patch(es) applied

  • Fix for tests using a combination of x:call[@template] and x:context (see pull request #3)
